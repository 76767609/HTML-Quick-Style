# AI HTML Page Generator

This project generates a simple HTML webpage using Python and Gradio.

## Features
- Generate HTML page from user input
- Apply color themes
- Simple website layout
- Live preview using Gradio

## Technologies Used
- Python
- Gradio
- HTML
- CSS

## How to Run
1. Run the Python file (app.py)
2. Enter title and content
3. Click submit
4. HTML page will be generated

## Output
A styled HTML webpage will be generated based on the user input.
